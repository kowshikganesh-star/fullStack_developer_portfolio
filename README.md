## 📝 How to Use

### 📦 Clone the repo
```bash
git clone https://github.com/kowshikganesh-star/java-developer-portfolio.git
cd java-developer-portfolio
```
### 📦 Install dependencies
```bash
npm install
```
### 🚀 Run the app
```bash
ng serve
```
### 🌐 Open in browser
```bash
http://localhost:4200

```
